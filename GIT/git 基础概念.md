# 基础概念
Git 的概念中分为工作区，暂存区，本地仓库，远程仓库的概念。 工作区，暂存区，本地仓库均是在本地电脑（个人电脑）上，这也就是所谓的分布式版本控制系统不需要网络也可以工作。远程仓库其实并不是一个地方，而是很多地方（除你自己外，所有的项目参与人员的本地仓库）。
-- 工作区 – 你建立的本地分支就是工作区
-- 暂存区 – 你在工作区的工作 git add后就添加到了暂存区
-- 本地仓库 – 当你把暂存区的文件git commit 后就提交到了本地仓库
-- 远程仓库 – 当你把本地仓库的文件 git push 后就是把本地仓库的修改推送到了远程仓库
